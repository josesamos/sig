Transformar e Integrar Información Geográfica

Ejercicios

Herramientas

Los ejercicios se podrán realizar con QGIS o R, o con una combinación de ambas, según las preferencias de cada persona.

Recomiendo realizarlas en R, entre otros motivos, porque es mucho más fácil (aunque pueda parecer que no).

Ejercicio 1. Capas del municipio

Para los datos de tu municipio asignado, transforma las capas de datos numeradas de lista del apartado 2.1, según las indicaciones del apartado 2.2.

Para documentar la realización del ejercicio:

  • genera un documento PDF

  • con un apartado para cada conjunto de datos numerado de lista del apartado 2.1

  • para cada capa generada, incluye una captura de pantalla completa de QGIS o RStudio donde se muestre:

    • el nombre de la capa y
    • la representación gráfica de la capa (mediante plot o mostrándola en QGIS) conjuntamente con la capa del contorno del municipio (no es necesario mostrar la tabla de datos ni sus metadatos). Para mostrar las capas se puede usar una herramienta distinta a la que se ha usado para obtenerlas (por ejemplo, se puede desarrollar en R y mostrala desde QGIS)

  • para las bandas de satélite, para documentar la realización del ejercicio, es suficiente con mostrar una de las bandas.

Los datos obtenidos en este tema deberán conservarse y estar disponibles para ser utilizados en las actividades que desarrollaremos en los temas siguientes.
